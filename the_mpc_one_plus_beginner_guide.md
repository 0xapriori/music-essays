## Part 1: Welcome to the MPC ONE+

### What Is the MPC ONE+?

The MPC ONE+ is a standalone music production center made by Akai Professional. It is a self-contained beat-making machine — no computer required. It has a 7-inch touchscreen, 16 velocity-sensitive pads, built-in synths, effects, a sampler, and a sequencer. You can create, mix, and export complete songs entirely on the device.

The MPC (Music Production Center) lineage goes back to 1988 with Roger Linn's original MPC60. The platform shaped the sound of hip-hop, R&B, pop, and electronic music for decades. The ONE+ carries that legacy into the modern era with a touchscreen workflow, Wi-Fi connectivity, and an expandable sound library.

### What Makes the ONE+ Different from the Original MPC ONE?

The MPC ONE+ (model ACVA2, released 2023) is an incremental update. Here's what changed:

| Feature | MPC ONE (Original) | MPC ONE+ |
|---|---|---|
| **Wi-Fi** | No | Yes — firmware updates, Splice integration, Ableton Link |
| **Bluetooth** | No | Yes — Bluetooth MIDI only (not Bluetooth audio) |
| **Internal Storage** | 4 GB (~2 GB usable) | 16 GB (~7.8 GB usable) |
| **Ethernet Port** | Yes | Removed (replaced by Wi-Fi) |
| **Color/Design** | Black/dark grey | Red/Crimson "Retro" colorway (also available in black) |
| **Built-in Speaker** | No | No |
| **Built-in Battery** | No | No |
| **Processor/RAM** | 1.8 GHz quad-core ARM / 2 GB | Same |
| **Pads, Knobs, I/O** | Identical | Identical |
| **Software/Firmware** | Same version line | Same version line |

The most meaningful upgrade is **Wi-Fi**, which enables wireless firmware updates, direct access to the expansion store, Splice sample sync, and Ableton Link for wireless tempo/transport sync. The quadrupled storage is also significant — more room for factory content and your own samples before needing external storage.

> **WARNING:** Bluetooth on the MPC ONE+ is **MIDI only**. You cannot connect Bluetooth headphones or Bluetooth speakers. You must use wired headphones or wired monitor speakers.

## Part 2: Know Your Hardware

### Top Panel Layout

#### The Pads (Center-Left)
- **16 velocity-sensitive, pressure-sensitive RGB pads** in a 4x4 grid
- Pads detect how hard you hit them (velocity) and continued pressure (aftertouch)
- RGB backlighting shows which sounds are loaded, active notes, and recording status
- **Pad Banks A–D** accessible via dedicated buttons; **Pad Banks E–H** via Shift + Pad Bank (or double-press)
- This gives you **128 pad slots total** per program (16 pads x 8 banks)

#### Q-Link Knobs (Above the Pads)
- **4 touch-sensitive endless encoders**
- Control parameters in real time: filter cutoff, volume, pan, tuning, effect parameters, etc.
- Press the **Q-Link button** to cycle through parameter columns
- The MPC detects when you touch a knob (before turning), enabling parameter display

#### Data Dial / Jog Wheel (Right Side)
- Large rotary encoder for scrolling through menus and adjusting values
- **Press it** to confirm/enter

#### +/- Buttons
- Fine-increment value adjustment for the selected parameter

#### Transport Controls (Below the Screen)
- **Play** — starts playback from current position
- **Play Start** — starts playback from the beginning of the sequence
- **Stop** — stops playback; press twice to return playhead to start
- **Record (Rec)** — arms recording; press then Play to begin recording
- **Overdub** — enables overdub mode (layers new notes without erasing existing ones)

#### Performance and Editing Buttons
- **Note Repeat / Latch** — hold + press a pad to trigger repeated hits at the current TC rate. Shift + Note Repeat latches it on
- **Full Level / Half Level** — pads trigger at max velocity (127). Shift activates Half Level (64)
- **16 Level / Notes** — spreads one pad's sound across all 16 pads with a parameter gradient (velocity, tune, filter, etc.)
- **Erase** — during playback, hold Erase + press a pad to delete that pad's notes in real time. When stopped, opens the Erase dialog
- **Copy / Delete** — copy pad settings from one pad to another
- **Undo / Redo** — undo last action. Shift + Undo = redo
- **Tap Tempo / Master** — tap in time to set BPM. Shift toggles master vs. per-sequence tempo
- **Shift** — held with other buttons for secondary functions
- **Menu** — opens the main menu. Shift + Menu opens the Browser directly

#### Touchscreen (7-inch)
- Full-color multi-touch capacitive display
- Supports tap, double-tap, swipe, and pinch-to-zoom
- Top toolbar shows: current mode, sequence info, BPM, time signature, bars, TC value, metronome, transport status
- Bottom edge shows context-sensitive action buttons

## Part 3: Initial Setup

### 3.1 Powering On for the First Time

1. Remove all packaging and the **protective film from the touchscreen** (easy to miss)
2. Place the unit on a stable, flat surface
3. Connect the 12V power supply to the rear DC jack and plug into a wall outlet
4. Flip the **power switch** on the rear panel
5. The unit boots to the main MPC interface in approximately 15–20 seconds
6. A blank default project loads automatically

### 3.2 Register Your Device

1. Go to [akaipro.com](https://www.akaipro.com) and create an account
2. Navigate to My Account > My Registered Products
3. Enter the serial number from the bottom of the unit
4. Registration unlocks software downloads, expansion packs, and Splice access

### 3.3 Firmware Update (Do This First)

**Method A — Wi-Fi Update (easiest for MPC ONE+):**
1. Tap Menu > Preferences (gear icon)
2. Connect to your Wi-Fi network under wireless settings
3. Tap UPDATE; if a new version is available, follow on-screen prompts

**Method B — USB Drive Update:**
1. Download the latest firmware (.img file) from akaipro.com/mpc-software-firmware-downloads
2. Place the file on a USB drive formatted to exFAT or FAT32
3. Connect the USB drive to one of the MPC ONE+'s USB-A ports
4. Tap Menu > Preferences > UPDATE > select "USB Drive Update" > tap UPDATE

**Method C — Computer Update:**
1. Connect MPC ONE+ to your computer via the included USB cable
2. Tap Menu > Preferences > UPDATE > select "Switch to Update Mode"
3. Launch the inMusic Software Center on your computer
4. Follow on-screen prompts

> **WARNING:** Keep the power supply connected throughout the entire update. Do not power off or use the device during the update. A failed firmware update can brick the device.

### 3.4 Audio Output Setup

- **Headphones:** Plug headphones into the front-panel 1/8" jack. Sound routes automatically. Adjust volume with the rear Master Volume knob.
- **Monitor Speakers:** Connect two 1/4" TRS cables from Outputs 1/2 on the rear panel to your powered monitors. Set speaker volume low initially, then adjust both the MPC Master Volume and speaker volume.
- **Both simultaneously:** The headphone output mirrors the main outputs. Both work at the same time.

### 3.5 Preferences and Settings

Tap Menu > Preferences to access:
- **Wi-Fi settings** — connect to your network
- **Bluetooth settings** — pair Bluetooth MIDI controllers (not audio devices)
- **Screen Brightness** — under Hardware settings
- **Pad Sensitivity** — adjust velocity curve (more on this in Part 10)
- **Audio Device** — select internal audio or a connected USB class-compliant audio interface
- **MIDI/Sync** — configure MIDI clock, MTC, Ableton Link sync

### 3.6 SD Card Setup

1. Buy a SanDisk Extreme 128 GB or 256 GB SDXC, Class 10 or higher
2. **Format to exFAT** on your computer before first use (or format within the MPC via Browser)
3. Insert into the SD card slot on the left side of the unit
4. Recommended folder structure on the SD card:

```
SD Card Root/
├── Projects/           (your saved MPC projects)
├── Samples/
│   ├── Drum Kits/
│   ├── Instruments/
│   ├── Loops/
│   ├── Vocals/
│   └── One Shots/
├── Expansions/         (for MPC expansion packs)
└── Synths/             (for synth presets)
```

> **WARNING:** Always **eject the SD card from the MPC browser** before physically removing it to avoid data corruption.

### 3.7 Loading Your First Sample Packs

The MPC ONE+ comes pre-loaded with approximately 2 GB of factory content including drum kits, instrument presets, loops, and one-shots. To load additional content:

- **Expansion Store:** With Wi-Fi connected, access the MPC expansion store directly on the device
- **From Computer:** Download expansions from akaipro.com, transfer to SD card or USB drive, place in the Expansions folder
- **Splice:** With Wi-Fi and a Splice subscription, sync curated sample collections wirelessly
- **Supported formats:** WAV (recommended), AIF/AIFF, FLAC, MP3, OGG

## Part 4: The MPC Mental Model (CRITICAL — Read Before Touching Pads)

This section is the single most important thing to understand before using the MPC. Most beginner confusion, frustration, and mistakes trace back to not understanding this hierarchy. Read it twice.

### The Hierarchy: Projects → Sequences → Tracks → Programs → Pads

```
PROJECT (the top-level container — one project = one song/session)
  │
  ├── SONG (an arrangement of sequences in order)
  │
  ├── SEQUENCES (building blocks / sections — up to 128 per project)
  │     │
  │     ├── TRACKS (layers within a sequence — up to 128 per sequence)
  │     │     │
  │     │     └── Each Track is assigned to one PROGRAM
  │     │
  │     └── Each Sequence has its own: length (bars), tempo, time signature
  │
  ├── PROGRAMS (instruments / sound sources — shared across ALL sequences)
  │     │
  │     ├── Drum Program → different sample on each pad (for kits)
  │     ├── Keygroup Program → sample pitched across pads (for melodic instruments)
  │     ├── Plugin Program → built-in synth engine
  │     ├── MIDI Program → controls external MIDI gear
  │     ├── CV Program → controls modular synths via CV/Gate
  │     └── Clip Program → for clip-launching workflow
  │
  └── SAMPLES (raw audio files loaded into the project's sample pool)
```

### Why This Matters

Here are the three things that confuse every beginner:

**1. Programs are project-wide, not sequence-specific.** If you load a drum kit as a Program, *every* sequence in the project shares that same kit. If you change a kick sound while working on your chorus (Sequence 3), that kick changes in your verse (Sequence 1) too. This is fundamentally different from a DAW where each track's instrument is independent.

**2. A Track is NOT a Program.** A Track is a container for MIDI note data (what you record when you hit pads). A Program is the instrument that generates sound. Tracks *point to* Programs. Multiple Tracks across different Sequences can point to the same Program.

**3. Sequences are patterns, not songs.** A Sequence is a short loop — typically 2–8 bars. Your verse is one Sequence. Your chorus is another Sequence. You chain them together in Song Mode. Stop trying to build an entire song inside one Sequence.

> **PRO TIP:** Think of it like a band. The **Programs** are the musicians (drummer, bassist, keyboardist). The **Tracks** are the sheet music for each musician. The **Sequences** are the sections of the song (verse, chorus). The musicians don't change between sections — they just play different sheet music.

### Audio Signal Path

```
Individual Pad (in a Drum Program)
  → Pad Channel Strip (volume, pan, 4 inserts, 2 sends)
    → Track Channel Strip (volume, pan, 4 inserts, 2 sends)
      → Submix Bus (optional grouping, 4 inserts)
        → Master Output (4 inserts)
          → Physical Outputs (Main Out 1/2 or Headphones)

Return Channels (fed by sends from any channel)
  → Return Channel Strip (volume, pan, 4 inserts)
    → Master Output
```

### Program Types Explained

**Drum Program** — The most common type. Each of the 128 pads (16 x 8 banks) holds a different, independent sample with its own volume, pan, tuning, filter, and effects. Use for: drum kits, chopped samples, one-shot sound effects.

**Keygroup Program** — Takes one or more samples and pitches them chromatically across the pads so you can play melodies and chords. Use for: bass lines, piano parts, pitched vocals, any melodic content built from samples.

**Plugin Program** — Uses one of the MPC's built-in synth engines (TubeSynth, Hype, Bassline, Electric, Mellotron, Solina, Odyssey, DrumSynth). No samples needed — sound is generated by synthesis. Use for: synth bass, pads, leads, keys, drum design.

**MIDI Program** — Sends MIDI data out of the MPC to control external hardware. Generates no sound on its own. Use for: sequencing external synths, drum machines, or other MIDI gear.

**CV Program** — Sends Control Voltage and Gate signals through the 3.5mm outputs to control modular synthesizers. Use for: sequencing Eurorack and semi-modular gear.

**Clip Program** — For Ableton-style clip launching in the Clip Launch view. Use for: live performance, non-linear jamming.

## Part 5: Essential MPC Vocabulary

### Core Structure

| Term | Definition |
|---|---|
| **Project** | The top-level container holding all sequences, tracks, programs, samples, and mixer settings. Saved as an .xpj file. |
| **Sequence** | A musical section (verse, chorus, intro) containing tracks. Up to 128 per project. Has its own bar length, tempo, and time signature. |
| **Track** | A single layer within a sequence assigned to one program. Contains recorded MIDI note data. Up to 128 per sequence. |
| **Program** | The instrument/sound source assigned to a track. Shared across all sequences in a project. |
| **Song** | An arrangement of sequences chained in order with repeat counts. |

### Pads and Performance

| Term | Definition |
|---|---|
| **Pad** | One of the 16 rubber buttons. Detects velocity (how hard you hit) and aftertouch (pressure). |
| **Pad Bank** | A group of 16 pads. Banks A–H give you up to 128 pad slots per program. |
| **Q-Link** | The four assignable rotary encoders above the pads. Control parameters in real time. |
| **Note Repeat** | Hold this button + a pad to retrigger it at a set rhythmic interval (1/8, 1/16, 1/32, etc.). |
| **16 Levels** | Spreads one pad's sound across all 16 pads with a parameter gradient (velocity, tune, filter). |
| **Full Level** | When on, all pad hits trigger at maximum velocity (127) regardless of strike force. |
| **Pad Perform** | Maps musical scales and chords to pads so every pad is "in key." |

### Timing and Rhythm

| Term | Definition |
|---|---|
| **BPM** | Beats Per Minute — the tempo of your sequence. |
| **Time Signature** | How beats group into bars. Default 4/4 = four beats per bar. |
| **Bar** | A measure of music. In 4/4, one bar = four beats. |
| **TC (Timing Correct)** | Quantization — snaps recorded notes to the nearest grid position (1/4, 1/8, 1/16, 1/32, etc.). |
| **Swing** | Shifts even-numbered subdivisions slightly late for a groovy, human feel. 50% = straight; 54–58% = classic MPC groove; up to 75% = heavy swing. |

### Sampling and Audio

| Term | Definition |
|---|---|
| **Sample** | A recorded piece of audio loaded into the MPC — drum hit, vocal phrase, chord, any sound. |
| **Chop** | To divide a longer sample into smaller segments that can be assigned to individual pads. |
| **Slice** | One segment created by chopping. Defined by start and end points within the original audio. |
| **Trim** | Setting the start and end points of a sample. Non-destructive — the original file is preserved. |
| **Loop** | A setting that makes a sample (or portion) repeat continuously between loop start and end points. |
| **Warp** | Time-stretching — change tempo without changing pitch, or vice versa. |
| **Autosampler** | Automatically samples a sound source across notes/velocities and builds a playable Keygroup program. |

### Recording and Editing

| Term | Definition |
|---|---|
| **Record** | Captures your pad hits as note data. **Replaces** existing content on the track. |
| **Overdub** | Records additional notes **on top of** existing content without erasing. |
| **Erase** | Removes recorded notes. Hold Erase + pad during playback for real-time deletion. |
| **Bounce / Mixdown** | Renders audio output to a new file (WAV, MP3). |
| **Export** | Renders your sequence or song as a stereo audio file for sharing. |

### Effects and Mixing

| Term | Definition |
|---|---|
| **Insert Effect** | Applied directly on a specific channel (4 slots per channel). Audio passes through completely. |
| **Send Effect** | On a shared bus — multiple tracks send varying amounts of signal to it (4 send buses). |
| **Master Effect** | Applied to the final stereo output — affects everything (4 slots). |
| **XYFX** | Touchscreen becomes an XY effects pad for real-time effect manipulation. |

## Part 6: Navigating the Touchscreen

### Main Mode (Home Screen)
The default view on boot. Shows current sequence, track, program assignment, and the 16-pad grid. This is your command center — you select tracks, sequences, and programs here. **How to access:** Press Menu > Main, or it loads on boot.

### Browser
File navigator for finding and loading samples, programs, expansions, and projects. Left panel shows Places (file system), Expansions (installed packs), and Search. Right panel shows file listings with preview. **How to access:** Press Shift + Menu.

### Grid View (Piano Roll)
Visual MIDI editor. Horizontal axis = time, vertical axis = pitch (or pad for drums). Tap to place notes, drag to move/resize, pinch to zoom. Select notes to cut, copy, transpose, or adjust velocity. **How to access:** Menu > Grid View or the Grid icon in the toolbar.

### Track View
All tracks in the current sequence displayed vertically with note data as blocks. Gives you a big-picture overview of your arrangement within a sequence. **How to access:** Menu > Track View.

### Step Sequencer
The 16 hardware pads become 16 step buttons (like a classic drum machine). Select a sound on screen, toggle steps on/off with pads. Ideal for precise, grid-based programming. **How to access:** Menu > Step Sequencer.

### Program Edit
Deep editing of the selected program. For Drum Programs: per-pad sample assignment, tuning, filter, envelope, layers. For Keygroup: sample mapping, velocity layers, key zones. For Plugin: synth parameters. **How to access:** Menu > Program Edit.

### Sample Edit
Waveform editor for trimming, looping, chopping, normalizing, reversing, and time-stretching samples. This is where you prepare raw audio before assigning to pads. **How to access:** Menu > Sample Edit.

### Channel Mixer
Full mixing console. Track Mixer shows channel strips for all tracks. Pad Mixer shows individual strips for each pad in a Drum Program (toggle via icon). Each strip has: volume, pan, mute/solo, 4 insert slots, send levels. Also shows submix groups, returns, and master bus. **How to access:** Menu > Channel Mixer.

### Song Mode
Arrange sequences into a complete song. Each row = one step with a sequence assignment and repeat count. Add, delete, reorder steps. Convert finished arrangement to a single sequence for final editing/export. **How to access:** Menu > Song Mode.

### Other Modes

| Mode | Purpose |
|---|---|
| **XYFX** | Touchscreen XY pad for real-time effect performance |
| **Pad Perform** | Musical scales/chords mapped to pads |
| **Looper** | Real-time audio looping |
| **Sampler** | Record audio from inputs or resample internally |
| **Sequence Editor (List Edit)** | Detailed event list showing every MIDI event with exact values |

## Part 7: Your First Beat — Step by Step

This is the heart of the guide. Follow these instructions exactly. Every button press and screen tap is described. We assume you have completed Part 3 (Initial Setup) and have headphones or monitors connected.

### 7.1 Creating a New Project

1. Power on the MPC ONE+ and wait for it to boot (~15–20 seconds)
2. A blank default project loads automatically with:
   - 1 Sequence (Sequence 001), 2 bars long
   - 1 Track (Track 01) assigned to a default Drum Program
   - Tempo of 120 BPM
3. To explicitly start fresh: tap **Menu** on the touchscreen, then tap **New Project**
4. **Save immediately:** Tap Menu > **Save As** > name your project (e.g., "My First Beat") > choose save location (SD Card recommended) > tap **Save**

> **PRO TIP:** Get in the habit of saving to your SD card, not internal storage. The internal storage fills up fast and is better reserved for factory content.

### 7.2 Loading Drum Sounds from the Browser

1. Press **Shift + Menu** to open the **Browser**
2. On the left side of the screen, tap **Expansions** to see all installed content packs
3. Tap any expansion thumbnail to browse its contents
4. Look for a folder called **Drum Programs** or **Kits**
5. Tap a drum kit name to preview it (you'll see its pads populate)
6. Tap **Load** (or double-tap the kit name) to load it into your project
7. The drum program is automatically assigned to Track 01
8. Close the Browser by tapping the **X** or pressing **Menu**

> **PRO TIP:** If you don't see expansions, try the **Places** tab instead. Navigate to Internal > Factory Content > Drum Programs to find the factory kits.

### 7.3 Understanding Your Drum Program and Pads

1. Back on the Main screen, you should see your loaded drum kit
2. **Hit the pads** — each pad triggers a different drum sound (kick, snare, hi-hat, etc.)
3. Hit pads **harder** for louder hits, **softer** for quieter — the pads are velocity-sensitive
4. The **Pad Bank** buttons let you switch between Banks A–D (Shift + Pad Bank for E–H)
5. Most drum kits put the essential sounds on **Bank A**:
   - Bottom row (Pads 1–4): Kick variations
   - Second row (Pads 5–8): Snare/clap variations
   - Third row (Pads 9–12): Hi-hats (closed, open) and cymbals
   - Top row (Pads 13–16): Percussion, FX, additional sounds

6. **Rename your track now:** Tap the Track name field on screen and type "Drums"

> **WARNING:** Different kits map sounds differently. Always hit every pad to learn where sounds are before recording.

### 7.4 Setting Your Tempo (BPM)

1. Look at the **top toolbar** on the touchscreen — you'll see the BPM value (default: 120.0)
2. **Method 1 — Tap the BPM field** on screen, then turn the **data dial** to adjust. Hold **Shift** while turning for fine increments (0.01 BPM)
3. **Method 2 — Tap Tempo:** Press the physical **Tap Tempo** button repeatedly in time with your desired groove. The MPC averages your taps.
4. **Method 3 — Direct entry:** Double-tap the BPM field to open a numeric keypad and type an exact value.

**Common tempos by genre:**
- Lo-fi / Boom bap: 75–95 BPM
- Trap: 130–170 BPM (often half-time feel)
- House / Techno: 120–130 BPM
- R&B: 60–80 BPM
- Pop: 100–130 BPM

For your first beat, try **90 BPM** — a comfortable hip-hop tempo.

### 7.5 Recording Your First Drum Pattern

#### Setting Up Before Recording

1. **Set the sequence length:** Tap the **Bars** field on the top toolbar. Turn the data dial to set it to **4** (four bars gives you room to breathe). Tap away to confirm.

2. **Set Time Correct (TC):** Tap the **TC** field on the toolbar (shows a note value like 1/16). Set it to **1/16** (sixteenth notes) — this means your pad hits will snap to the nearest sixteenth note, keeping things tight. For your first recording, having TC on is your safety net.

3. **Turn on the metronome:** Tap the **metronome icon** on the toolbar (it looks like a small metronome or speaker). You should hear a click on each beat when you press Play. Adjust metronome volume using the Q-Link knob assigned to it, or in the metronome settings.

#### Recording: Pass 1 — Kick and Snare

4. Press the **Record (Rec)** button — it will blink, indicating recording is armed
5. Press **Play** — you will hear a **one-bar count-in** (four clicks), then recording begins
6. During the count-in, get your timing. On the downbeat (beat 1), **hit the kick pad**
7. Play a simple kick/snare pattern. For your first time, try this classic pattern:

```
Beat:  1   .   .   .   2   .   .   .   3   .   .   .   4   .   .   .
Kick:  X   .   .   .   .   .   .   .   X   .   .   .   .   .   .   .
Snare: .   .   .   .   X   .   .   .   .   .   .   .   X   .   .   .
```

8. When the sequence reaches the end of 4 bars, it **automatically loops back** and switches to **Overdub mode** (the Overdub button lights up)
9. You can keep playing to add more notes on subsequent passes, or press **Stop** when you're satisfied

> **WARNING:** If you press **Record + Play** again after stopping (with Overdub OFF), you will **erase everything** and start fresh. Always check that the **Overdub button is lit** before recording additional layers. This is the #1 beginner mistake.

#### Understanding Record vs. Overdub

- **Record mode** (Rec button lit, Overdub NOT lit): **Replaces** everything on the track. Use only when you want to start over.
- **Overdub mode** (Overdub button lit): **Adds** new notes on top of existing ones. Use for layering.

### 7.6 Quantizing Your Performance (Timing Correct)

If you had TC set to 1/16 before recording, your notes were auto-quantized as you played. But if you recorded with TC off (or want to adjust), here's how to quantize after the fact:

1. Tap the **TC** field on the toolbar to open the **Time Correct window**
2. Set the **Time Division:** 1/16 is standard for most beats. Use 1/8 for simpler patterns, 1/32 for detailed hi-hat work.
3. Set **Swing:** 50% = perfectly straight. Try **54–58%** for a classic MPC groove feel. This shifts every other sixteenth note slightly late, creating a human bounce.
4. Set **Strength:** 100% = full snap to grid. Try **75%** to preserve some of your natural feel while tightening timing.
5. Tap **Do It** to apply

> **PRO TIP:** The MPC's swing implementation was designed by Roger Linn and is the secret weapon behind countless classic hip-hop records. Experiment with swing percentages — even small changes (52% vs. 56%) dramatically affect the feel. Start with 54%.

### 7.7 Overdubbing — Adding More Drums

Now let's layer in hi-hats and percussion:

1. Make sure the **Overdub button is lit** (press it if it's not)
2. Press **Play** — your kick/snare pattern plays back
3. While the sequence loops, **hit the hi-hat pad** in rhythm. Try steady eighth notes (2 hits per beat) or sixteenth notes (4 hits per beat)
4. On the next pass, add a percussion element — a shaker, a tambourine, a rim shot
5. **Made a mistake?** Hold the **Erase** button + press the offending pad as the playhead passes over the bad notes — they'll be deleted in real time
6. Press **Stop** when you're happy

### 7.8 Using Note Repeat for Hi-Hats

Note Repeat is the fastest way to program hi-hat patterns:

1. Make sure **Overdub** is on and press **Play**
2. Hold the **Note Repeat** button
3. While holding Note Repeat, **press and hold a hi-hat pad** — the MPC fires repeated hits at the current TC rate
4. While still holding Note Repeat, tap the **time division buttons** on screen to change the rate:
   - **1/8** = eighth notes (standard)
   - **1/16** = sixteenth notes (driving)
   - **1/32** = thirty-second notes (rolls/fills)
   - **1/8(3)** or **1/16(3)** = triplets (bouncy, shuffle feel)
5. **For hands-free operation:** Press **Shift + Note Repeat** to **latch** it on. Now you can release the Note Repeat button and it stays active. Press it again to turn it off.
6. After recording a sustained note-repeat hi-hat pattern, open the **Grid View** to selectively remove hits and create your actual pattern

### 7.9 Adding a Bass Line (Keygroup or Plugin Program)

Time to add a melodic element on a new track.

#### Creating a New Track

1. On the Main screen, look at the **Track** field in the upper area
2. Tap the **right arrow** next to the track name (or turn the data dial) to move to **Track 02**
3. A new empty track appears

#### Option A: Using a Plugin Synth (Recommended for Beginners)

4. Tap the **program type icons** (the row of small icons near the track name). Select the **Plugin** icon (looks like a waveform or synth)
5. A list of available synth plugins appears: **TubeSynth, Bassline, Hype, Electric, Mellotron, Solina, Odyssey**
6. For a bass line, select **Bassline** (designed for bass sounds) or **Hype** (versatile)
7. Browse the **presets** — tap through them and hit pads to preview. The pads now play chromatically (Pad 1 = C2, Pad 2 = C#2, Pad 3 = D2, etc.)
8. Find a bass sound you like and tap **Load** or **Select**

#### Option B: Using a Keygroup Program

4. Tap the **Keygroup** program type icon (looks like a keyboard)
5. Open the Browser (Shift + Menu), navigate to a bass sample (a single sustained bass note)
6. Load it — the MPC automatically pitches it across all pads chromatically

#### Recording the Bass Line

9. **Rename this track:** Tap the track name and type "Bass"
10. Make sure **Overdub** is on
11. Press **Record**, then **Play** (or just Play if Overdub is already on)
12. Play your bass line on the pads. The bottom-left pads are lower notes, moving chromatically up and to the right
13. Press **Stop** when done

> **PRO TIP:** Enable **Pad Perform** mode for easier melodic playing. Press Menu > Pad Perform, select a scale (e.g., C Minor), and now every pad plays only notes in that scale — no wrong notes possible.

### 7.10 Adding a Melody or Chords

Repeat the process from 7.9 to add more melodic elements:

1. Move to **Track 03**
2. Create a new **Plugin Program** — try **Electric** for a Rhodes/keys sound, **Hype** for a synth pad, or **Mellotron** for vintage textures
3. Browse and select a preset
4. Use **Pad Perform** mode with your desired scale for easy chord playing:
   - **Scale mode:** Each pad = one note in the scale
   - **Chord mode:** Each pad = a full chord from the scale
5. Record with Overdub on, playing over your existing drums and bass

### 7.11 Basic Mixing — Volume and Pan

1. Tap **Menu > Channel Mixer** (or the mixer icon in the toolbar)
2. You'll see channel strips for each track (Drums, Bass, Melody, etc.)
3. **Volume:** Drag each track's fader up/down to balance levels. Your drums and bass should be the loudest elements.
4. **Pan:** Tap the pan knob on each strip and adjust. Keep kick, snare, and bass **centered (C)**. Pan hi-hats, percussion, and melodic elements slightly left or right for width.
5. **Mute/Solo:** Tap the **M** button to mute a track, **S** to solo it (hear only that track)
6. **Pad Mixer:** For fine control over individual drum sounds, tap the **Pad Mixer** icon (looks like a single pad). This shows individual faders for each pad in your Drum Program — perfect for balancing kick vs. snare vs. hi-hat volumes.

> **PRO TIP:** Use the **Q-Link knobs** for hands-on mixing. Press the Q-Link button to cycle through parameter columns — one column controls the volume of tracks 1–4, another controls pan, etc.

### 7.12 Copying Sequences and Building Arrangement

Your 4-bar beat is your foundation. Now let's create variations:

1. On the Main screen, make sure you're on **Sequence 001**
2. Tap **Menu > Sequence** (or tap the pencil icon next to the sequence name)
3. Select **Copy Sequence**
4. Set Source: Sequence 001, Destination: Sequence 002
5. Tap **Do It** — Sequence 002 is now an exact copy
6. Navigate to Sequence 002 (tap the sequence field and turn the data dial)
7. **Create a variation:** For example, mute the hi-hats, add a new percussion element, or change the bass line
8. Repeat to create Sequences 003, 004, etc. — an intro (stripped down), a chorus (full energy), a breakdown, etc.

> **PRO TIP:** Name your sequences immediately. Tap the sequence name and type descriptive names: "Verse," "Chorus," "Intro," "Bridge." This saves massive confusion later in Song Mode.

### 7.13 Song Mode — Arranging Your Beat

1. Tap **Menu > Song Mode**
2. You'll see a list where each row represents a step in your song
3. **Method 1 — Manual entry:** Tap the first row, use the data dial to select which sequence plays (e.g., "Intro"), set the repeat count (e.g., 2x). Move to the next row, select the next sequence ("Verse"), repeat count (4x), and so on.
4. **Method 2 — Real-time recording:** Press **Record** in Song Mode, then press **Play**. As the song plays, tap pads to trigger sequence changes (each pad can correspond to a sequence). This is great for quickly sketching a structure.
5. A typical arrangement:

```
Step 1: Intro      x2  (8 bars)
Step 2: Verse      x4  (16 bars)
Step 3: Chorus     x2  (8 bars)
Step 4: Verse      x4  (16 bars)
Step 5: Chorus     x2  (8 bars)
Step 6: Bridge     x2  (8 bars)
Step 7: Chorus     x4  (16 bars)
Step 8: Outro      x2  (8 bars)
```

6. Press **Play** to audition your full song arrangement
7. **Optional:** Tap **Convert to Sequence** to flatten the entire song into one long sequence for final editing and export

### 7.14 Exporting Your Beat (Bounce/Mixdown)

1. Tap **Menu > Export** (or Menu > Save > Export Audio Mixdown)
2. Configure settings:
   - **Source:** Select your Song (or the Sequence you want to export)
   - **Start Bar / End Bar:** Define the range
   - **Audio Tail:** Set to at least **2 seconds** (captures reverb/delay tails)
   - **File Format:** WAV (recommended for quality) or MP3
   - **Bit Depth:** 16-bit/44.1kHz (CD quality) or 24-bit/44.1kHz (production quality)
3. Choose save location (SD Card recommended)
4. Tap **Do It** to render

#### Exporting Stems (Individual Tracks)

For transferring to a DAW for further mixing:
1. Follow the same export path
2. Check the **"Explode Tracks"** option
3. This renders every track as a separate audio file — extremely useful for collaboration or mastering in a full DAW

> **PRO TIP:** Always export at 24-bit WAV if you plan to do any further processing. You can always convert to MP3 later, but you can't add quality back.

## Part 8: Sampling — The Soul of the MPC

Sampling is what made the MPC legendary. It's the practice of recording audio — from records, instruments, YouTube, field recordings, anything — and transforming it into something new.

### What Sampling Is and Why It Matters

At its core, sampling means taking a piece of existing audio and using it as raw material. You might sample a 2-bar drum break from a vinyl record, chop it into individual hits, rearrange them on pads, and create a completely new beat. Or sample a vocal phrase and pitch it into a melody. The MPC was built for this workflow.

### Recording Audio into the MPC

#### From an External Source (Turntable, Synth, Phone, etc.)

1. Connect your audio source to the rear **Audio Inputs 1/2** using 1/4" cables
2. Tap **Menu > Sampler** to open the Sampler screen
3. Set the **Input Source** to Input 1/2
4. Adjust the **Record Gain** knob on the rear panel — watch the input meter on screen. You want the signal peaking around -6dB to -3dB (loud but not clipping/red)
5. Set the **Monitor** level to hear the input through your headphones
6. Press the **Record** button on screen (or arm recording and set a threshold for auto-start)
7. Play your source material
8. Press **Stop** when done
9. The recorded sample appears in your project's sample pool

> **WARNING:** For turntables: you need a **phono preamp** between the turntable and the MPC. The MPC's inputs are line-level, not phono-level. For guitars: use a **DI box or preamp** since guitar output is instrument-level.

#### Resampling (Recording the MPC's Own Output)

1. Open the Sampler
2. Set Input Source to **Resample** or **Main Out**
3. Press Record, then Play your sequence — the MPC records its own audio output
4. Great for bouncing tracks, capturing live knob tweaks, or creating new samples from your existing work

### Basic Sample Editing

1. Tap **Menu > Sample Edit**
2. Your sample's waveform displays on screen
3. **Trim:** Drag the **Start** and **End** markers to remove silence or unwanted audio at the beginning/end
4. **Loop:** Tap the Loop settings to set loop start/end points for sustained sounds
5. **Normalize:** Boosts the sample's volume to maximum without clipping
6. **Reverse:** Flips the sample backwards
7. **Fade In/Out:** Smoothly ramp volume at the start or end

### Chopping Samples

This is the MPC's superpower — slicing a longer sample into pieces and spreading them across pads.

#### Auto Chop (Threshold-Based)

1. In Sample Edit, tap **Chop**
2. Set the mode to **Threshold**
3. Adjust the **Threshold** slider — lower values detect quieter transients, higher values only catch loud hits
4. The MPC automatically places slice markers at detected transients (drum hits, note attacks)
5. Tap **Play** on individual slices to preview them
6. **Manually adjust:** Tap and drag any slice marker to fine-tune its position

#### Manual Chop

1. In Chop mode, set the mode to **Manual**
2. Play through the sample, and tap **+ marker** at each point where you want a chop
3. Fine-tune positions by dragging markers

#### Assigning Chops to Pads

1. After chopping, tap **Convert** (or Shift + Convert)
2. Select **"New Drum Program Using Slices"**
3. Each slice is assigned to its own pad in a new Drum Program
4. Go to Main Mode, select the new program, and play your chops on the pads — rearrange, flip, and remix the original audio

> **PRO TIP:** When adjusting chop points, edit only the **END point** of each region. This automatically fixes the start of the next region, preventing gaps or overlaps. This is the technique experienced MPC users rely on.

## Part 10: Common Mistakes and How to Avoid Them

### 1. Recording in Replace Mode Instead of Overdub

**MISTAKE:** You press Record + Play to add hi-hats over your kick/snare, and the kick/snare gets erased.

**WHY IT HAPPENS:** Record mode replaces everything on the track. Overdub mode adds to it. The distinction isn't immediately obvious, and the buttons are right next to each other.

**THE FIX:** Always check that the **Overdub button is lit** before recording additional layers. If you accidentally overwrite, immediately press **Undo**. Make Overdub your default mentality — you should almost never use Record mode after your first pass.

---

### 2. Not Understanding Programs Are Shared Across Sequences

**MISTAKE:** You change a drum sound while working on your chorus, then go back to your verse and find the sound changed there too.

**WHY IT HAPPENS:** Programs are project-level resources, not sequence-specific. All sequences share the same pool of programs.

**THE FIX:** If you need a different version of a sound in a different sequence, create a **new Program** (duplicate the original and rename it) and assign it to the track in that sequence. See Part 4 for the full mental model.

---

### 3. Not Saving Projects Regularly

**MISTAKE:** Hours of work lost after a crash, accidental power-off, or unintended operation.

**WHY IT HAPPENS:** No prominent "unsaved changes" indicator like in desktop software.

**THE FIX:** Enable **Auto-Save immediately**: Menu > Project > Auto Save, set to 5-minute intervals. Also manually save (Menu > Save) frequently. Always save before powering off.

---

### 4. Working Entirely on Internal Storage

**MISTAKE:** Internal storage fills up, can't save projects or load new content.

**WHY IT HAPPENS:** The internal drive seems like the default place to work, and there's no upfront warning about its ~7.8 GB limit.

**THE FIX:** Buy an SD card on day one and use it as your primary working drive for projects and samples. Reserve internal storage for factory content.

---

### 5. Confusing Drum Programs and Keygroup Programs

**MISTAKE:** Loading a Keygroup when you want a drum kit, or a Drum Program when you want a melodic instrument.

**WHY IT HAPPENS:** Both use samples, but they organize them completely differently.

**THE FIX:** **Drum Program** = one different sample per pad (for kits, chops, one-shots). **Keygroup Program** = one sample pitched chromatically across all pads (for bass, keys, melodic instruments). See Part 4.

---

### 6. Forgetting to Switch Automation from Write to Read

**MISTAKE:** After recording automation (Q-Link knob movements, fader changes), every subsequent playback records unwanted parameter changes because automation is still in Write mode.

**WHY IT HAPPENS:** The automation record state (red = write, green = read) isn't visually prominent, and there's no auto-disarm after recording.

**THE FIX:** After every automation recording pass, immediately switch from **Write (red)** to **Read (green)**. Check this before every playback.

---

### 7. Over-Quantizing Everything

**MISTAKE:** Setting TC to 1/16 at 100% strength and leaving it on permanently. Beats sound mechanical and lifeless.

**WHY IT HAPPENS:** Quantization feels like a safety net, and "tight" seems like it should be good.

**THE FIX:** Experiment with recording with TC off and quantizing after the fact at **reduced strength (50–75%)**. The slight timing imperfections are what give MPC beats their legendary "swing" and groove. Add **Swing (54–58%)** for even more human feel.

---

### 8. Setting Pad Sensitivity Too High

**MISTAKE:** Every tap registers near max velocity. Beats are flat and dynamically dead.

**WHY IT HAPPENS:** Higher sensitivity seems like it should mean "more responsive." It's counterintuitive that less sensitivity produces more expressive playing.

**THE FIX:** Go to Preferences > Pad Sensitivity and **lower it**. You want a comfortable hard hit to register 127 and a light tap to register 40–60. This gives you dynamic range and groove.

---

### 9. Not Naming Tracks, Programs, and Sequences

**MISTAKE:** After 30 minutes, everything is "Track 01," "Program 001," "Sequence 01." The mixer shows Program names, the Track Mute view shows Track names — total confusion.

**WHY IT HAPPENS:** Renaming on a touchscreen feels tedious.

**THE FIX:** Name everything **immediately** when you create it. Tracks: "Drums," "Bass," "Keys." Sequences: "Intro," "Verse," "Chorus." Programs: "Trap Kit 01," "Sub Bass." This is non-negotiable for productive work.

---

### 10. Leaving Pad Edit Mode on "All" Instead of "Current"

**MISTAKE:** Editing one pad's parameters (envelope, filter, tuning) silently applies changes to all pads, creating a mess that's hard to undo.

**WHY IT HAPPENS:** The mode indicator is small and easy to forget.

**THE FIX:** Always check the pad edit mode indicator before making changes. Set it back to **"Current"** as soon as you're done with multi-pad editing.

---

### 11. Trying to Use the MPC Like a DAW

**MISTAKE:** Expecting a visual timeline overview, clip launching, drag-and-drop, and familiar DAW workflows. Getting frustrated when the MPC doesn't work that way.

**WHY IT HAPPENS:** The touchscreen looks like a small DAW, but the underlying paradigm is fundamentally different.

**THE FIX:** Accept that the MPC is a **performance instrument** with its own logic. Build short sequences, chain them in Song Mode, use Track Mutes for variation. The constraint is the feature — it forces creative focus.

---

### 12. Sloppy Sample Chopping

**MISTAKE:** Auto-chop creates slices that don't start exactly on transients. Chops have tiny gaps or bleed from previous sounds.

**WHY IT HAPPENS:** Auto-chop with default settings doesn't always nail every cut point.

**THE FIX:** After auto-chopping, **always manually review** each slice's start point. Edit only the **END point** of each region (this fixes the start of the next). Lower the threshold to catch quieter hits.

---

### 13. Not Planning Sequence/Track Layout Before Starting

**MISTAKE:** Diving straight in without structure. Ending up with a mess of tracks and no room for expansion.

**WHY IT HAPPENS:** The MPC lets you start immediately, so planning feels unnecessary.

**THE FIX:** Before recording a single note, set up your first sequence with all the tracks you'll need: Drums, Bass, Chords, Lead, FX, plus 2–3 empty tracks. Name everything. When you duplicate this sequence for variations, the organization carries over.

---

### 14. Ignoring the Shift Button

**MISTAKE:** Struggling with imprecise adjustments, missing hidden features, not knowing about alternate functions.

**WHY IT HAPPENS:** Shift modifier functions aren't labeled on buttons.

**THE FIX:** Shift is the most powerful button on the MPC:
- **Shift + Data Dial** = fine resolution (0.1 increments)
- **Shift + Q-Link** = fine knob adjustment
- **Shift + Note Repeat** = latch Note Repeat
- **Shift + Menu** = open Browser directly
- **Double-tap any value field** = numeric keypad entry

---

### 15. Can't Get Past the 8-Bar Loop

**MISTAKE:** Making loops that sound great but never finishing a full song.

**WHY IT HAPPENS:** The jump from loop to arrangement feels overwhelming.

**THE FIX:** Use this workflow: (1) Build your main loop as Sequence 1. (2) Copy it to Sequences 2, 3, 4. (3) Create variations using Track Mutes (strip drums for intro, add elements for chorus). (4) Chain in Song Mode. (5) Done. Arrangement doesn't have to be complicated.

## Part 11: Pro Tips and Workflow Hacks

### The Shift Button Mastery

1. **Shift + Data Dial** = Fine resolution on any parameter (0.1 level in mixer, 0.01 semitone in tuning, 0.01 BPM)
2. **Shift + Q-Link** = Fine knob adjustment
3. **Double-tap any value field** = Opens numeric keypad for precise direct entry
4. **Shift + Note Repeat** = Latch mode (stays on without holding)
5. **Shift + Menu** = Open Browser directly

### Note Repeat Tricks

6. Use Note Repeat for **instant hi-hat patterns**: Latch it on (Shift + Note Repeat), record a full bar of 1/16th notes, then go into Grid View and delete specific hits to create your pattern. Faster than programming each hit individually.
7. **Change rates mid-recording**: While Note Repeat is active, switch between 1/8, 1/16, and 1/32 to create natural-sounding fills and variations.

### 16 Levels Power

8. Press **16 Levels**, select a hi-hat pad, choose **Velocity** as the parameter. Now all 16 pads play the same hi-hat at 16 different velocities — record a pattern and you'll get instant dynamic variation.
9. Select **Tune** as the parameter to create a melodic line from a single percussion sample.

### Mixing Shortcuts

10. **Q-Links during mixing**: Press and hold the **Q-Link button** at any time to see current assignments. The bottom Q-Link in column 1 often controls TC value.
11. **Solo a pad within a Drum Program**: Open the Pad Mixer and use the Solo button per pad to isolate individual drum sounds.

### Arrangement Hacks

12. **Track Mutes for easy variation**: Instead of creating a completely new sequence for every section, duplicate your main sequence and use **Track Mutes** (Menu > Track Mute or the mute buttons in the mixer) to strip elements in/out. Mute the drums for an intro, unmute everything for the chorus.
13. **Next Sequence mode**: During playback, change which sequence plays next by tapping the sequence field and selecting a different one. The switch happens at the end of the current loop — great for live performance.

### Sample Workflow

14. **Resampling trick**: Want to commit effects or create a new starting point? Set the Sampler to **Resample** mode, play your sequence, and record the output. Now you have a finished audio bounce you can chop further.
15. **Quick sample from Browser**: While browsing samples, tap to preview. If you like it, hold **Shift and drag** (or use the Load button) to drop it directly onto a pad.

### Time-Savers

16. **Copy Events between sequences**: Instead of re-recording, copy MIDI data from one track/sequence to another. Menu > Sequence > Copy Events. Select source and destination.
17. **Undo is your best friend**: The MPC has deep undo history. Press **Undo** immediately if anything goes wrong. Don't panic.
18. **Save As with new name before experimenting**: Before making risky changes, Save As with a new name (e.g., "My Beat v2"). Now you can experiment freely with a safe fallback.

### Sound Design

19. **SP-1200 effect for grit**: Apply the SP-1200 insert effect to any track for instant 12-bit lo-fi character. Classic hip-hop texture.
20. **Layer sounds on one pad**: In Program Edit for a Drum Program, each pad supports up to 4 sample layers. Stack a kick with a sub-bass sample for thickness.

## Part 12: The Learning Roadmap

### Week 1: Foundation

**Goal:** Understand the core structure and make your first beat.

Focus on:
- Hardware layout (pads, transport buttons, Q-Links, touchscreen navigation)
- The **Project > Sequence > Track > Program** hierarchy (Part 4 — this is critical)
- Loading a drum kit from factory content

- Setting BPM
- Basic quantization (TC at 1/16)
- Saving projects to SD card
- Using the Pad Mixer to adjust drum volumes

### Week 2–3: Expand Your Toolkit

**Goal:** Add melody, learn sampling basics, start arranging.

Focus on:
- **Plugin Programs** — load TubeSynth, Electric, or Hype. Browse presets. Play melodies on pads.
- **Pad Perform** mode — set a scale so every pad is "in key"

- **Chopping** — load a loop, auto-chop, convert slices to a Drum Program
- **Note Repeat** for hi-hat patterns
- **Insert effects** — reverb on snare, compressor on drums
- **Send effects** — shared reverb bus
- **Grid View** — manually edit note positions, lengths, velocities
- **Song Mode** — chain multiple sequences into an arrangement
- **Swing** — apply 54–58% swing and hear the difference

### Month 2+: Advanced Features

Focus on:
- **Keygroup Programs** in depth — multi-sample instruments, velocity layers
- **Autosampler** — clone external synths into self-contained MPC instruments
- **MIDI Programs** — sequence external hardware
- **CV Programs** — control modular synths
- **XYFX** — real-time effect performance

## Part 13: Connectivity Scenarios

### Scenario 1: Standalone with Headphones (Simplest)

- Plug headphones into the front-panel 1/8" (3.5mm) jack
- Power on, start making beats
- Adjust volume with the rear Master Volume knob
- **What you need:** Just headphones with a 3.5mm plug (or adapter)
- **Best for:** Getting started, portable use, quiet environments

### Scenario 2: Standalone with Monitor Speakers

- Connect two 1/4" TRS cables from Outputs 1/2 to your powered monitors
- Set monitor volume low initially, then adjust MPC Master Volume and speaker volume
- **What you need:** Powered studio monitors + two 1/4" TRS cables
- **Best for:** Serious production, accurate sound reference

### Scenario 3: Connected to Computer (Controller Mode)

- Connect via USB-B cable to your computer
- Install MPC 2 Desktop Software and the appropriate driver
- On the MPC, go to Menu > Preferences and enter Controller Mode
- The MPC acts as a MIDI controller for MPC 2 software; audio routes through your computer
- **Important:** In controller mode, the MPC's internal drive is not visible in the browser; it shows your computer's drives. SD cards still mount.
- **Important:** Firmware versions must match between hardware and MPC 2 software
- **Best for:** Using VST/AU plugins, larger screen editing, recording into a DAW

### Scenario 4: With External MIDI Synths

- **MPC → Synth:** MIDI cable from MPC MIDI Out to synth's MIDI In. Create a MIDI Program track, set output to the synth's MIDI channel.
- **Synth → MPC:** MIDI cable from synth's MIDI Out to MPC MIDI In. Configure in Preferences > MIDI/Sync.
- **USB MIDI:** Connect class-compliant USB MIDI controllers to either USB-A port. Use a powered USB hub if connecting multiple devices.
- **Sync:** MIDI Clock, MTC, or Ableton Link (over Wi-Fi)

### Scenario 5: With External Audio Sources

- Connect external audio output to MPC Audio Inputs 1/2 using 1/4" cables
- Adjust Record Gain knob on the rear panel
- Use the Sampler to record/sample incoming audio
- **For turntables:** You need a phono preamp between the turntable and MPC
- **For guitars:** You need a preamp or DI box (guitar output is instrument-level)
- Turn down speakers and monitor through headphones when recording to avoid feedback

### Scenario 6: With Modular Synths (CV/Gate)

- Use 1/8" (3.5mm) patch cables from the MPC's four CV/Gate outputs to your modular gear
- Configure CV/Gate settings in the track settings (create a CV Program)
- The MPC can send pitch CV, gate, and clock signals
- **Best for:** Eurorack and semi-modular synth users
